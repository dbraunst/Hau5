Hi there! Thanks for checking out "Hau5" v0.0.1

It's still a super early experiment, but it's a fun thing that I hope you enjoy! 

~~~~~~~~~~~~
DESCRIPTION: 
~~~~~~~~~~~~

A fun interactive way to mess with a pre-composed electronic piece that I (Daniel Braunstein) wrote. Music + Visuals inspired by electronic musician Deadmau5.


~~~~~~~~~~~~
INSTALLATION
~~~~~~~~~~~~
 - On Windows:
	 - Download zip file (ending in x86_64)
	 - Extract zip file
	 - Run 'Hau5_0.0.1.exe'
 - On OSX: 
	 - Download zip file (ending in .OSX)
	 - Extract zip file
	 - Run Application 'Hau5_0.0.1_OSX

~~~~~~~~~~~~~
INSTRUCTIONS: 
~~~~~~~~~~~~~

To start/stop piece: 
 - SpaceBar!


To control soundcubes:
 - W E R / Y U I 
	 - these move the cubes up / between layers. 
 - S D F / H J K
	 - these move the cubes down / between layers. 

**for ease of use, position hands on either side of the "T" and "G" keys. Upper row moves up, Lower row moves down. 


To leave experience: (I'll be adding buttons in the future, pinky swear)
 - On Windows: Alt+f4
 - On Mac: Command + Q
